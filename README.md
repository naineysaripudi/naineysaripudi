### 📈 Stock Movement Prediction System

> A 6-model ensemble system trained on years of historical stock data for next-day trend classification, deployed via an interactive dashboard.

- 🧠 Ensemble of Logistic Regression, SVM Polynomial, Random Forest, XGBoost, Voting Ensemble & LSTM
- 📊 Trained on 2,685 trading days of INFY.NS data
- 🛠️ 14 engineered technical indicators (RSI, MACD, momentum, volatility, moving averages)
- 🔁 Walk-forward time-series cross-validation (5 folds) to prevent data leakage
- 🏆 Best F1 score of 0.654 achieved by SVM Polynomial with custom threshold optimization
- 📉 Real-time trend visualization via interactive Streamlit dashboard

`Python` `TensorFlow` `Scikit-Learn` `XGBoost` `Streamlit`
